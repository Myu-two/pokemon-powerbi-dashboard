# Pokémon Interactive Dashboard

## 📌 Overview

This project is an interactive Pokémon dashboard developed using Power BI, focusing on data cleaning, transformation, modeling, and visualization.

The objective of this project was to explore a Pokémon dataset obtained from Kaggle and transform raw data into interactive and meaningful visual insights.
